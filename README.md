🚀 This project is a frontend clone of the Amazon homepage built using HTML and CSS.
It focuses on replicating a real-world e-commerce user interface and improving frontend development skills.

📌 Description
This project demonstrates how to build a modern e-commerce homepage layout similar to Amazon.
It includes a navigation bar, hero section, and product cards designed with clean UI principles.

⚙️ Tech Stack
HTML5
CSS3
Flexbox
Responsive Design


✨ Features
✔️ Amazon-like homepage UI
✔️ Responsive layout
✔️ Navigation bar
✔️ Hero banner section
✔️ Product cards section
✔️ Clean and structured code

📚 What I Learned
Building real-world UI layouts
Using Flexbox for alignment
Writing clean and organized HTML & CSS
Improving UI/UX design understanding
Structuring large frontend projects
